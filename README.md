## 📖 Usage

1. Browse components
2. Copy the code
3. Paste into your React project
4. Customize as needed

Example:

```jsx
import React from "react";

export default function Button() {
  return (
    <button className="px-4 py-2 bg-black text-white rounded-lg">
      Click Me
    </button>
  );
}
```
